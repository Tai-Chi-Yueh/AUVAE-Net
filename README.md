# AUVAE-Net
Code for Adversarial U-shaped Variational Autoencoder Network (AUVAE-Net) —— A Deep Learning Method for Hyperspectral Image Unmixing in Remote Sensing
# Environment
* Windows
* python 3.10.16
* pytorch 1.12.1
* cudatoolkit 11.2.2 + cudnn 8.1.0.77
* keras 2.10.0
* numpy 1.23.5
* tensorflow-gpu 2.10.0
